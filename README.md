## A Propos
DraftBot est le bot officiel du [serveur discord](https://www.draftman.fr/discord) du développeur DraftMan.
Il est développé en NodeJS et contient de nombreux modules :
- Discord.js
- Youtube API
- Fortnite API
- Twitter API

## Commandes
Il y a de nombreuses commandes:
 - google
 - infos
 - commande
 - fortnite
 - help
 - ping
 - quote
 - say
 - clear
 
 - music play
 - music skip
 - music pause
 - music resume
 - music queue
 - music np
 - music volume
 - music clear
 
 
## Config

```
{
  "prefix": "!",
  "youtube_api": "YOUR_KEY",
  "token": "YOUR_TOKEN"
}
```
